üõ† Available Tools

Core Tools

  • run_glif: Run a glif with specified ID and inputs
  • glif_info: Get detailed information about a glif including input fields

Discovery Tools Default ON

  • list_featured_glifs: Get a curated list of featured glifs
  • search_glifs: Search for glifs by name or description
  • my_glifs: Get a list of your published glifs
  • my_glif_user_info: Get detailed information about your account

Metaskill Tools Default ON

  • save_glif_as_tool: Save a glif as a custom named tool
  • remove_glif_tool: Remove a saved glif tool
  • remove_all_glif_tools: Return to pristine state
  • list_saved_glif_tools: List all saved glif tools

Bot Tools Beta

  • list_bots: Get featured bots and sim templates
  • load_bot: Get detailed bot information including skills
  • save_bot_skills_as_tools: Save bot skills as individual tools
  • show_bot_info: Get specific bot information
